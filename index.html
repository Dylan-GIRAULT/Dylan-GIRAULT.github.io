Space Conquest

  • Each agent runs in its own goroutine and communicates via a message-passing model to negotiate alliances
  • The project integrates a combinable strategy system across four axes (observation, preparation for launch, response and alliances)
  • Interactive 3D visualization of the solar system via ReactThreeFiber, allowing real-time tracking of the simulation's evolution and comparison of agent behaviors in a non-deterministic environment
React.js TypeScript Yarn Go
